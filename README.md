# shodan
Uses shodan command line interface to make requests to shodan API 

### Run Shodan

```bash
$ git clone https://github.com/arlidge/Shodan.git
$ cd Shodan/
python3 -m pip install shodan

$ chmod +x shodan.sh
$  ./shodan.sh
```
